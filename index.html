Genetic Algorithm Simulation
This is a 3D agent simulation experiment.

It was done to play with state machines, genetic algorithms, the Rust programming language in a web context, and the Three.js library.

The simulation is based on a wolf-sheep ecosystem, with grass as a food source.

The stats of the agents can be observed by clicking on them, and also through the graph viewer which will log the stats of the agents over time.
